## 这是个什么游戏

这是一个基于 **Kaboom.js + Vite** 的像素冒险小游戏项目，当前从 `kaboom-vibe-workshop` 课程参考模板开始迭代。

现在的世界变成了致敬 **The Binding of Isaac** 的房间制射击冒险：蓝色方块在迷宫房间里移动，用方向键向四个方向发射子弹，清掉所有红色敌人后，绿色门才会开启。

## 当前玩法

```text
你 = 蓝色方块，负责移动、探索、撞出结果
墙 = 灰色墙体，会限制移动和挡住子弹
敌人 = 红色方块，会自由移动，碰到它们就失败
子弹 = 蓝色小弹幕，用方向键发射，打中红色方块会清掉敌人
门 = 绿色方块，清空房间敌人后出现，碰到它进入下一房间
终点 = 通过全部三个房间
```

## 操作说明

| 按键 | 效果 |
|------|------|
| `W` | 向上移动 |
| `S` | 向下移动 |
| `A` | 向左移动 |
| `D` | 向右移动 |
| `↑` | 向上射击 |
| `↓` | 向下射击 |
| `←` | 向左射击 |
| `→` | 向右射击 |
| `R` | 胜利或失败后重新开始 |

## 胜负条件

清掉当前房间的全部红色敌人后，绿色门出现。

碰到绿色门，进入下一房间。

碰到移动的红色敌人，失败，然后按 `R` 再来。

通过全部三个房间，完成通关。
