## 这是个什么游戏

这是一个基于 **Kaboom.js + Vite** 的像素冒险小游戏项目，当前从 `kaboom-vibe-workshop` 课程参考模板开始迭代。

现在的世界变成了致敬 **The Binding of Isaac** 的西游主题房间制射击冒险：孙悟空会穿过火焰山、白骨洞、流沙河、女儿国、盘丝洞、狮驼岭前哨、小雷音寺前庭等西游节点，用方向键向四个方向发射金箍棒弹幕，清掉当前房间妖怪后，四周可用的绿色传送门才会开启。

## 当前玩法

```text
你 = 孙悟空，负责移动、探索、清理房间
墙 = 灰色墙体，会限制移动和挡住子弹
敌人 = 按场景变化的妖怪，会自由移动，碰到它们会扣生命
精英妖怪 = 狮驼岭前哨中的大型敌人，需要多发子弹击败
精英血条 = 多血量敌人头顶会显示分段血量，命中后实时减少
精英怒吼 = 狮驼岭精英会短暂蓄力预警，随后小幅加速一段时间
伏击房 = 特定房间清掉第一波敌人后会追加精英伏击，击破伏击后才会开门
子弹 = 金箍棒点射，用方向键发射，每次攻击后需要短暂蓄力，打中妖怪会清掉敌人
门 = 绿色传送门，清空房间敌人后按房间配置在四周出现，碰到它进入对应相邻房间
门名提示 = 开门后每个传送门旁会显示目标房间名，特殊房会带宝 / 精 / 终标记
出口预告 = HUD 会在开门后列出方向到目标房间，帮助快速决定下一步路线
回血桃 = 受伤后首次清空普通房间时可能出现，拾取后恢复 1 点生命
分身毫毛 = 狮驼岭前哨清空后掉落，拾取后射击会追加一枚偏移副弹
芭蕉扇 = 龙宫宝库首次探索时可获得，拾取后射击变为扇形三发弹幕
百宝谱 = 宝库三选一道具会从同类西游宝物池中抽取，名称不同但效果归入双发 / 扇形 / 护身三类
龙宫宝库 = 非战斗奖励房，探索分支里可领取一次攻击道具或回血奖励
终点 = 探索多房间网络并击破小雷音寺前庭的黄眉 Boss
小地图 = 右上角显示当前节点、已探索/已清空节点、当前出口方向和房间类型标记
开始路线图 = 开始菜单按房间坐标展示西游路线节点，方便开局前理解分支结构
81 难题材池 = 开始菜单会展示后续可扩展的西游劫难题材预告，当前不改变实际房间数量
房间短句 = 进入房间时短暂显示西游主题文本，帮助识别当前场景
劫难节点 = 每个房间带有“第 X 难”标记，把路线包装成西游 81 难中的关键节点
清房短评 = 战斗房清空后会显示一句西游主题短评，不打断开门和探索
地面装饰 = 房间有轻量主题纹理，例如熔岩裂纹、沙纹、水纹、蛛网和金光
入场安全期 = 每次进房间后短暂免伤，方便看清局面
用时 HUD = 右上角实时显示本局用时，帮助玩家感知评级压力
攻击状态 = 左侧 HUD 会显示攻击就绪或蓄力倒计时，方便掌握点射节奏
清房进度 = 状态栏显示已清空房间数 / 总房间数，帮助判断探索完成度
暂停面板 = 暂停时显示当前房间、生命、清房进度、用时和道具状态
评分 = 通关后根据用时和受伤次数给出评级，阈值已按 7 房间与精英房流程校准
```

## 操作说明

| 按键 | 效果 |
|------|------|
| `W` | 向上移动 |
| `S` | 向下移动 |
| `A` | 向左移动 |
| `D` | 向右移动 |
| `↑` | 向上射击 |
| `↓` | 向下射击 |
| `←` | 向左射击 |
| `→` | 向右射击 |
| `I / K / J / L` | 上 / 下 / 左 / 右射击替代键 |
| `P` | 暂停 / 继续，并查看操作提示 |
| `M` | 静音 / 开启音效 |
| `R` | 游戏中从第 1 房间满血重开；胜利或失败后也可重新开始 |

## 胜负条件

清掉当前房间的全部红色敌人后，房间四周可用的绿色门出现。

如果当前生命不满，首次清空普通房间时会出现回血桃，拾取后恢复 1 点生命。

碰到绿色门，进入对应方向连接的西游主题房间；已清空房间返回时不会重复刷怪。

清空房间开门后，每个传送门旁会显示目标房间名，奖励房、精英房和终点房会带“宝 / 精 / 终”标记，减少多出口选择时的迷路感。

右上角小地图会显示当前房间、已探索 / 已清空节点和当前房间出口方向，并用“宝 / 精 / 终”标记奖励房、精英房和终点房。

不同房间带有轻量主题地面装饰，只作为视觉氛围，不影响碰撞、子弹或移动。

每个房间入口会显示对应的西游“第 X 难”节点，例如第 47 难火焰山借扇、第 76 难狮驼岭妖军、第 81 难小雷音黄眉。

战斗房清空后会短暂显示一句西游主题短评，让开门反馈更有文化味。

狮驼岭前哨是第一个精英房雏形，里面的大型精英妖怪需要多次命中才会被击破。

精英妖怪头顶会显示分段血条，方便判断当前剩余生命。

狮驼岭精英会周期性短暂停顿并显示“怒吼蓄力”，随后小幅加速一段时间，注意拉开距离。

黄眉 Boss 进入半血阶段后会触发“金钹合鸣”，并略微提高移动速度。

清空狮驼岭前哨后会掉落攻击道具“分身毫毛”，拾取后当前一局射击变为轻量双发，但仍受统一攻击间隔限制。

女儿国分支连接到龙宫宝库，宝库无战斗，首次进入且没有攻击道具时会给“芭蕉扇”，让射击变为扇形三发，但仍需要按攻击间隔点射；已有道具时会按生命状态给回血桃或“宝库已取”提示，回访不会重复发放。

碰到移动的红色敌人会损失 1 点生命，并获得短暂无敌时间。

每次进入房间后有短暂入场安全期，生命只剩 1 点时会出现低血量提示。

生命归零后失败，然后按 `R` 再来。

右上角会实时显示本局用时，通关后会根据用时和受伤次数结算评级。

状态栏会显示清房进度，方便判断当前一局已经净化了多少房间。

暂停时会显示当前房间、生命、清房进度、用时和道具状态，方便中途规划路线。

击破终点房间小雷音寺前庭的黄眉 Boss，完成通关。通关后会显示用时、击败数、受伤次数、剩余生命、持有道具、清房进度、评级和本地最快记录；评级阈值已按多房间流程校准。
