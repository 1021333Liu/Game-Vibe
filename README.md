## 这是个什么游戏

这是一个基于 **Kaboom.js + Vite** 的像素冒险小游戏项目，当前从 `kaboom-vibe-workshop` 课程参考模板开始迭代。

现在的世界变成了致敬 **The Binding of Isaac** 的西游主题房间制射击冒险：孙悟空会穿过火焰山、白骨洞、流沙河等主题房间，用方向键向四个方向发射金箍棒弹幕，清掉所有妖怪后，绿色传送门才会开启。

## 当前玩法

```text
你 = 孙悟空，负责移动、探索、清理房间
墙 = 灰色墙体，会限制移动和挡住子弹
敌人 = 按场景变化的妖怪，会自由移动，碰到它们会扣生命
子弹 = 金箍棒弹幕，用方向键发射，打中妖怪会清掉敌人
门 = 绿色传送门，清空房间敌人后出现，碰到它进入下一房间
终点 = 通过全部三个房间
```

## 操作说明

| 按键 | 效果 |
|------|------|
| `W` | 向上移动 |
| `S` | 向下移动 |
| `A` | 向左移动 |
| `D` | 向右移动 |
| `↑` | 向上射击 |
| `↓` | 向下射击 |
| `←` | 向左射击 |
| `→` | 向右射击 |
| `R` | 胜利或失败后重新开始 |

## 胜负条件

清掉当前房间的全部红色敌人后，绿色门出现。

碰到绿色门，进入下一房间。

碰到移动的红色敌人会损失 1 点生命，并获得短暂无敌时间。

生命归零后失败，然后按 `R` 再来。

通过全部三个房间，完成通关。
