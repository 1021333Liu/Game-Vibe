## 这是个什么游戏

这是一个基于 **Kaboom.js + Vite** 的像素冒险小游戏项目，当前从 `kaboom-vibe-workshop` 课程参考模板开始迭代。

现在的世界还很小：一个蓝色方块、一个绿色目标、几个会移动的红色障碍。但游戏开发就是这样，从能跑开始，再一步一步变得更好玩。

## 当前玩法

```text
你 = 蓝色方块，负责移动、探索、撞出结果
目标 = 绿色方块，碰到它就胜利
障碍 = 红色方块，会自由移动，碰到它们就失败
```

## 操作说明

| 按键 | 效果 |
|------|------|
| `W` / `↑` | 向上移动 |
| `S` / `↓` | 向下移动 |
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `R` | 胜利或失败后重新开始 |

## 胜负条件

碰到绿色方块，胜利。

碰到移动的红色方块，失败，然后按 `R` 再来。
